<h1 align="center">MLP</h1>

## Description

 Implementation of a multi-layer perceptron using only NumPy.<br>
 Approximates the function:

    f(x_1, x_2, x_3, x_4) = (x_1^2, 3x_2, 5x_4 - x_3, 3)

## Features
- Linear layers, ReLU activation, MSELoss
- Training with SGD, weight decay (L2 regularization), gradient clipping, gradient accumulation
- Dataset generation and train/validation split
- Unit tests for all components
